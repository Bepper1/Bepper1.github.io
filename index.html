Bitcoin Explanation Proposals


Why this webiste


Bitcoin is really hard to understand, but I'm trying. Attempting to explain something in a variety of ways is a really good way to learn, so here goes. I'll try to explain bitcoin concepts using different analogies. Hopefully, some of these will help others to improve their own understanding. If I'm really lucky, other people will point out my failures so that I too can improve.


If you want to help out


I'm pretty new to git and github, but I'll try to maintain these pages from the associated repository. So comment there if I've messed something up. Just don't try to sell me on another project. I've explored that road enough to exhaust my enthusiasm, and will stick with BTC from here on out.

Please keep in mind that I'm expressing my opinions here. If you want to see a change, all you have to do is change my mind. Maybe we can all learn something in the process.
